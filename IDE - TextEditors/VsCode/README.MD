VS Code / Cursor
=======


Find references
---------------



Similar to Pycharm "Find usage"





**Cursor-specific features**

* AI-powered search: Use Cmd/Ctrl+K to ask about usages semantically
* Codebase search: Use Cmd/Ctrl+Shift+F for text search across files

**Quick access**

* Right-click a symbol → "Go to References" or "Find All References"
* Or use the keyboard shortcuts above

Shortcut -- Hotkeys
===================


* F5: Launch -- Run file
* ctrl+alt+f5 (custom for python): run current file

* CTRL+SPACE: Show suggestions
* Shift+Alt+F: Format file

* Go to Definition (F12)
* Peek Definition (Alt+F12)
* Find All References: Shift+F12
* Peek References (Alt+Shift+F12)

* [Go (code navigation)](https://code.visualstudio.com/docs/languages/go#_code-navigation)

* F12: Go definition
* Alt+F12: Peek Definition
* Shift+F12: Show ALl refs
* Shift+Alt+H: Show calls from or to a func
* CTRL+F12: GO to implementation
