Create a Complete 2D Survivors Style Game in Godot 4
=====================================================

**_Notes_**

* [Create a Complete 2D Survivors Style Game in Godot 4](https://www.udemy.com/course/create-a-complete-2d-arena-survival-roguelike-game-in-godot-4/)



* [EditorFeatureProfile](https://docs.godotengine.org/en/stable/classes/class_editorfeatureprofile.html)
* [Godot 4 Editor Feature Profiles](https://www.youtube.com/watch?v=3rIGegAQ810)

* Editor -> Manager Editor Features -> Create Profile -> disable 3D features 



* **Free Assets!!!** https://kenney.nl/

For pixel art you should set Texture FIlter from Linear -> Nearest

* Project - > Project Settings -> Rendering -> Texture -> Default Texture filter from Linear to Nearest


#### 2D Viewport size

* Width: 640 
* Height: 360
* Window Width Override: 1920   | 1600  | 1280
* Window Height Override: 1080  | 900   |720
* Stretch mode: viewport


**Remove Pixel Jittering**

* Project -> Rendering -> 2D -> Snap 2D Transform to Pixel
